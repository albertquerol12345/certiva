# Plan General de Contabilidad (PGC) — Resumen práctico

## Grupos de cuentas relevantes

- **Grupo 6 (Compras y gastos)**: recoge los consumos necesarios para la actividad. Ejemplos habituales: 600 (Compras), 621 (Arrendamientos), 628 (Suministros), 629 (Otros servicios).  
- **Grupo 7 (Ventas e ingresos)**: refleja la facturación. 700 (Ventas de mercaderías), 705 (Prestaciones de servicios), 706 (Descuentos sobre ventas).  
- **Cuenta 472 / 477**: IVA soportado / IVA repercutido. El 472 se carga con el IVA soportado deducible y el 477 se abona con el IVA repercutido en ventas.
- **Cuenta 410 / 400**: Proveedores a corto plazo; **430 / 440**: Clientes a corto plazo.

## Principios básicos

1. **Devengo**: se registra cuando se produce el hecho económico, no cuando se cobra/paga.  
2. **No compensación**: no se compensan saldos de cuentas salvo autorización legal.  
3. **Imagen fiel**: los estados deben reflejar la realidad económica sin sesgos.

## Clasificación habitual de facturas

| Tipo de factura | Cuenta gasto/ingreso recomendada | Notas |
| --------------- | -------------------------------- | ----- |
| Suministros (luz, agua, internet) | 628 | Se acompaña de IVA soportado (472). |
| Alquiler oficinas | 621 | Atender retenciones de arrendamientos si aplica. |
| Software/Servicios cloud | 629 / 627 | Depende del uso recurrente vs. proyectos. |
| Ventas de servicios | 705 | IVA repercutido (477). |
| Ventas intracomunitarias | 705/708 + IVA 0% (mencionar nir/nif-vies). |

## Referencias rápidas

- **Notas de crédito / abonos**: se invierten los signos (Debe ↔ Haber).  
- **Intracomunitarias**: requieren inversión del sujeto pasivo y prueba de transporte.  
- **Bienes de inversión**: si superan el umbral, amortización en 2xx/28x.
